# HTML 标签组成和关系

# HTML 标签的结构

  1. 标签由 <,> , / 与英文单词或字母组成，并把标签中 <> 包括起来的英文单词或字母成为标签
  2. 常见标签由两部分组成，称为双标签，前部分叫开始标签，后部分叫结束标签，两部分之间包裹内容
  3. 少数标签由一部分组成，我们称之为：但标签，自成一体，无法包裹内容
