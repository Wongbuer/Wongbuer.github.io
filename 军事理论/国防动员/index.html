# 国防动员摘要

# 一、 国防动员的概念及特征

# 1. 概念

国家发生战争或严重危机时，采取紧急措施，将社会各部门的资源进行整合集中，统一协调，分配，并将国家体制由平时转入战时，以应对危机，服务战争.

# 2. 特征

  1. 时间上 ——“连续性”

  2. 范围上 ——“全面性”

  3. 程序上 ——“严格性”

  4. 意义上 ——“非替代性”

# 二、 国防动员的功能

# 1. 应战功能

  1. 止战 —— 为遏止可能的战争提供威慑力量

  2. 制战 —— 为争夺战争主动权提供有效手段

  3. 胜战 —— 为赢得战争提供实力保障

# 2. 应急功能

  1. 应对不可抗力 —— 严重的自然灾害

  2. 应对突发事件 —— 公共卫生事件、暴恐事件

# 3. 服务功能

保障国家发展，支持经济建设

① 节约国防开支，集中力量发展经济

② 国防动员成果直接服务于经济社会

# 三、 国防动员的基本领域

# 1. 人民武装动员 —— 核心

国家为适应战争需要，将军队由平时体制转为战时体制的行为

具体措施:

  1. 现役部队迅速补充满员

  2. 把预备役部队转为现役

  3. 地方部队升级为野战部队

  4. 组织新作战部队

# 2. 国民经济动员

为应对战争和突发事件的需要，国家将经济部门及其相应体制有组织、有计划地从平时转入战时状态而采取的措施.

# 3. 交通战备动员

国家统一管制各种交通路线、设施、工具和通信系统，在全国或部分地区调集交通、通信力量，全力保障战争和处置突发事件需要的应急行动.

# 4. 人民防空动员

指国家为适应战争需要，发动和组织人民群众防备敌人空袭，减少空袭损失，消除空袭后果所进行的活动

# 5. 科技动员

国家展示统一组织调整科学相关部门和专家、工程技术人员，从事战争所需科学技术的开发研究，研制新式武器装备的措施.

# 6. 政治动员

国家从政治上、组织上、思想上发动人民和军队参加战争或应对危机所采取的措施

形式:

  1. 发布政治动员令

  2. 进行国防教育

  3. 媒体宣传

  4. …

# 四、 国防动员的类型

# 1. 动员规模 —— 总动员 & 局部动员

总动员：在全国范围内实施全方位的动员

局部动员：在部分地区或全国范围内部分资源的动员

# 2. 动员实施方式 —— 公开动员 & 秘密动员

公开动员：国家宣布进入战争状态

秘密动员：为避免暴露战略意图，伪装实施掩护

# 3. 动员实施的时间 —— 战前动员 & 后续动员

战前动员：战争即将爆发时进行的动员

后续动员：为补充战争消耗 & 增强作战能力
