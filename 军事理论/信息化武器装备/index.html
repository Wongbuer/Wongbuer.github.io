# 翼龙 Ⅱ 军用无人机

# 性能参数简介

最大飞行速度 : 370km/h

最小飞行速度 : 150km/h

最大飞行高度 : 9000m

任务续航时间 : 20h

起飞滑翔距离 : 1000m

着陆滑翔距离 : 1200m

视距链路半径 : 200km

# 在军事领域的应用情况

翼龙无人机不仅具备对敌目标进行精确打击的能力，还能够携带侦察设备对敌方目标进行远距离长航时侦察，总体性能已经达到了国际上同类型无人机的先进水平。依此数据来看，该机性能酷似美军著名的 “捕食者” 无人攻击机，‘捕食者’无人机以在伊拉克和阿富汗制造 “斩首行动” 而闻名。据了解，该机不仅性能不亚于美军 “捕食者”，而且价格却远比对方低。

# 发展前景展望

  1. 高空长航型

    拥有长滞空能力的无人机，在战场上空长时间提供情报和支援的情形在美国电影中已经多次出现。达到高空的无人机必须拥有一定的材料硬度和抗干扰能力，能在强电磁干扰的高空环境清晰与地面传达信息、接受指令，同时还能根据地面形势的变化给地面人员提供支援 [5]。不仅如此，战场的形势还要求这些无人机能够以较短的起降时间和起降距离进行起降，以适应多样化的任务需求。因此，大型无人机作战平台将是未来军用无人机的重要发展趋势。

  2. 微小型

    未来战场，特种作战必不可少，微小型无人机也是未来的一个发展方向。这种无人机不但便于携带，而且具有较强的单兵作战能力 [6]。微小型无人机要求在功能多样的同时成本可控，是一个单兵支援平台，能在较短的时间内为士兵提供准确的战场信息。例如，美军即将装备的一种微型无人机，无人机全长仅 16.7 厘米，重量仅为 28.3 克，完全可以轻松的放入到口袋里，并且能够抵近目标侦查而完全不用担心被发现，其拍照清晰程度也完全不逊于大型无人侦察机。
