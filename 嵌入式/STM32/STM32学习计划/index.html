# STM32 学习计划

# 放假之前

学到 OLED 与 LCD

小项目：实现在 LCD 屏中打开与切换图片 (图片通过串口发送) 实现一个定时开关 LED (串口发送定时时长) 按键也可调节定时时长

# 七月二十号

学到 SPI 通信

小项目：I2C SPI 协议通信实例 基于 LCD 显示的温度传感器

# 七月底

学到 OV2640 摄像头

小项目：正点原子教程视频 OV2640 实例自己做出来以及之前视频实例

# 八月十号

学号录音

小项目：制作录音机项目 视频实例

# 八月二十号

学完 STM32 开始做项目

项目：整合录音 播放歌曲 查看照片 (可提前将图片转码放在代码里 也可通过串口发送) 红外遥控 所有功能都可通过触屏实现 若已经实现 可以在屏幕中加入查看 OV2640 摄像头功能 触屏可以开关蜂鸣器 LED 等基本外设
