# 宣讲会

# 导语

大一的同学在初期学习 C 语言的时候可能觉得自己编写出来的程序只有一个黑框框，自己能够做的也只是在这个黑框框中输入输出，没有什么精良的画面。但你们可不要因此对整个编程世界产生这样的偏见。也许同学们都渐渐体会到了，学习嵌入式的过程，和其他过程不同的是，嵌入式学习能够时刻反映出你所打的每一行代码，对这个现实世界产生的影响。因为你的一行代码，开发板上的 LED 灯点亮了，或者是蜂鸣器响了…… 在嵌入式的学习过程中，你绝对会比别人更加清楚 “代码” 的分量.

那么我要给同学们讲的也是一个团队的一个基础项目，不过不要看是基础项目，但是项目本身的立意与取得的成果，都是值得称赞的

# 基于人工智能的火灾预测系统

火灾一直是人类的敌人，不仅使人类遭受重大损失，也使一些我们消防员失去了他们英勇的生命。防患于未然，于是我们就想着，能不能在酿成火灾的初期就及时的发现并采取行动呢。所以我们创造了一个基于人工智能的火灾预测系统

首先，我们曾经做的原型机长这样，虽然看起来很丑很普通，外表像一个快递盒，但是通过这个机器，可以检测到附近的比如 CO2, CO, 液化石油气之类的有毒有害气体并实时监测温度和湿度。采集到的数据将不断地送往后端，通过人工智能模型研判后，将大楼的耐火时间，人员的逃生时间显示出来，以供消防人员进行更加迅速与精准的抢救.

同学们可能会疑惑，这也要人工智能？同学可能对实际情况的预估太乐观了。就像同学们调侃的，你永远不知道你的队友在用什么设备打游戏一样，你也永远不知道人们会在烟雾报警器下做出什么。我们要做的，就是排除寝室里因为吃自热火锅 (火锅) 导致的过多烟雾而误报的问题，或者失恋的男同学在烟雾报警器下抽烟而导致的误报问题，这些举动有导致火灾的风险，同学们应该避免，但是我们要做的就是解决这些同学们的无意之举导致的假性警报，而提供哪些真正的已发威胁.

这些是我们通过这个项目曾经获得的奖项。同学们可以了解一下，物联网的学习过程，还是能取得很多成绩的.

同学们可能会觉得这个系统除了人工智能的部分，其他部分的水平似乎没有到达那种高大上的程度。确实是这样，但是在我看来，嵌入式方向的学习，不仅是学习技术，也是一个提升自己的建设心与责任感的过程。也许所用到的技术并不高级，但是只要你想到了，有想法了，我们所学习的知识，是真的可以切切实实地、显著的、深刻的影响着人们的生活。大家可以想象一下，救护车护送病人的时候，过往红绿灯能联动着做出优化，可能挽救一条生命；通过城市中四处部署的温湿度模块与天气预报系统，城市天气预报的精度指数级上升，大气这个混沌系统，至少在这个城市，我们可以做到分钟级、或者秒级的预报；还有智能家庭……

扯远了，大家可以看到我们这个系统用到了一些传感器，正是这些传感器，让你代码中的输入，不只是 scanf, 而是多个维度，像人的各个感官一般，接受着信息，而我们要做的，就是为这些传感器赋予智能，让它们的数据真正改变人们的生活，直到人们习以为常，感觉不到它们的存在.
